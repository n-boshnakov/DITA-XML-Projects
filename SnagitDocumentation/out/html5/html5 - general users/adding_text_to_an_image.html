Adding Text to a Screenshot

You can add information to a captured image by writing on it.

You have completed Printing your Screen.

  1. Select Text.
  2. Drag and drop a text box on your screenshot.
  3. Modify the text style.
    • Change the font.
    • Change the font size.
    • Change the font fill.
  4. Write the text.
  5. Save your file.
